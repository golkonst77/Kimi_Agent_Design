# Документация проекта

## Быстрый старт
1. `npm install`
2. `npm run dev`
3. Открыть `http://localhost:5173`

## Команды
- `npm run dev` — локальная разработка.
- `npm run build` — production‑сборка.
- `npm run lint` — линтинг.

## Архитектура и соглашения
- `src/components/site` — публичные секции сайта.
- `src/admin` — админ‑панель.
- `src/contexts` — состояние и обновление контента.
- `src/data/defaultContent.ts` — исходные данные контента.
- Именование: компоненты `PascalCase`, хуки `useX`, файлы `PascalCase.tsx`.

## Внешние сервисы и зависимости
- React Router — маршрутизация.
- Radix UI — UI‑примитивы.
- Sonner — уведомления.
- Яндекс.Метрика — аналитика (загружается только после согласия).

## Переменные окружения
- `VITE_YANDEX_METRIKA_ID` — ID счетчика Яндекс.Метрики. Без значения баннер не отображается.

## Версионирование
- Источник версии: `package.json`.
- Версия отображается в футере через `__APP_VERSION__` (см. `vite.config.ts`).
- Каждое изменение должно сопровождаться обновлением версии и записью в `README.md` → `Changelog`.
