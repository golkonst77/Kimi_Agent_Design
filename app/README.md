# Kimi Agent Design

Публичный сайт и админ‑панель для студии/дизайнера интерьеров. Проект охватывает витрину услуг, отзывы, проекты и базовую CMS для управления контентом.

## Цель проекта
Сформировать премиальный образ бренда, дать быстрый доступ к кейсам и услугам, а также сократить время обновления контента через админ‑панель.

## Ключевые задачи
- Презентация услуг и кейсов с акцентом на конверсию.
- Быстрое обновление контента без участия разработчика.
- Четкая навигация: "Проекты" — основной источник демонстрации работ.

## Техническое и бизнес‑позиционирование
- Технически: SPA на React + Vite, админ‑панель и контент через контекст.
- Бизнес: премиальный дизайн интерьеров, индивидуальный подход, доверие через кейсы и отзывы.

## Версия
Текущая: v0.1.3 (источник — `package.json`, отображается в футере).

## Структура проекта
- `src/components/site` — публичные секции сайта.
- `src/admin` — админ‑панель и страницы управления контентом.
- `src/contexts` — контекст и состояние сайта/проектов/авторизации.
- `src/data` — дефолтный контент.
- `src/types` — типы доменной модели.
- `docs/README.md` — онбординг, стандарты, команды.

## Основные разделы/фичи
- Главная, обо мне, услуги, отзывы, контакты.
- Проекты (список и детальная страница).
- Админ‑панель для управления контентом.

## Используемые технологии
- React 19, TypeScript, Vite, React Router, Tailwind CSS.
- UI‑компоненты Radix + кастомные обертки.

## Рабочие сценарии
- `npm run dev` — локальная разработка.
- `npm run build` — production‑сборка.
- `npm run lint` — проверка линтинга.
