## Аналитика (Яндекс.Метрика)

### Как включить метрику
1. Создай счетчик в Яндекс.Метрике.
2. В корне проекта добавь `.env`:
   ```
   VITE_YANDEX_METRIKA_ID=12345678
   ```
3. Перезапусти dev‑сервер.

### Как работает согласие
Баннер согласия показывается только если указан `VITE_YANDEX_METRIKA_ID`.
Скрипт метрики загружается только после согласия пользователя.

### Где находится логика
- Баннер: `src/components/site/CookieConsent.tsx`
- Роут политики: `/privacy`

### Сценарии
- **Согласился** → метрика подгружается.
- **Отклонил** → метрика не грузится.
- **Не показали баннер** → переменная окружения не задана.
