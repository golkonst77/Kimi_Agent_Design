## Развертывание

Ниже приведены способы запуска проекта локально, сборки и деплоя.

### Требования
- Node.js 18+ (рекомендуется 20+).
- npm 9+.

### Локальный запуск
1. Установить зависимости:
   `npm install`
2. Запустить dev‑сервер:
   `npm run dev`
3. Открыть:
   `http://localhost:5173`

### Production‑сборка
1. `npm run build`
2. Проверить сборку локально:
   `npm run preview`

### Деплой на любой статический хостинг
Проект — SPA на Vite, можно размещать на Netlify, Vercel, Cloudflare Pages,
GitHub Pages, Nginx и т.д.

Шаги:
1. Выполнить `npm run build`.
2. Залить содержимое `dist/` на хостинг.
3. Убедиться, что роутинг SPA настроен на отдачу `index.html` для любых путей.

### Переменные окружения
Создай файл `.env` в корне:
```
VITE_YANDEX_METRIKA_ID=12345678
```
Если переменная не задана, баннер согласия и метрика не отображаются.

### Проверка после деплоя
- Открывается главная.
- Работают страницы `/projects` и `/project/:id`.
- Открывается `/privacy`.
- Метрика загружается только после согласия в баннере.
