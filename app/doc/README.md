## Документация проекта

Этот каталог содержит подробные инструкции по развертыванию и адаптации
сайта под конкретную дизайн‑студию или клиента.

### Содержание
- `deployment.md` — установка, сборка, деплой.
- `customization.md` — адаптация проекта под клиента.
- `content-editing.md` — редактирование контента через админ‑панель.
- `analytics.md` — подключение Яндекс.Метрики с согласием.
- `troubleshooting.md` — типовые проблемы и решения.
